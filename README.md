# BinStorm Example

## Installation

### 1. Clone project and install the dependencies
```
git clone git@github.com:binbytes/binstorm-example.git && cd binstorm-example && npm install
```

### 2. Run Project

For Android

```
npm run dev-android
```

For iOS

```
npm run dev-ios
```
